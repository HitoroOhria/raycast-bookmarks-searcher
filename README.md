# Bookmarks Searcher

Raycast command that search bookmarks from browser.

![Image](https://github.com/user-attachments/assets/bbaaecd3-2cd8-4b98-9562-056be4c80f89)

## Development

build and use.

```shell
$ npm i
$ npm run dev
```

run format.

```sehll
$ npm run format
```

run lint.

```sehll
$ npm run lint
```
